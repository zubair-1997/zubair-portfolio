Project Title: CAD Model Manipulation in Mixed Reality

In this project, we created a CAD model using Autodesk Fusion 360, which was then imported into Unity to implement an interaction technique. Designers and engineers often face challenges when interacting with 3D models in mixed and virtual reality. To address this issue, we developed an interaction technique that simplifies model manipulation. Using hand gestures, users can easily move the model from one location to another. Additionally, each model features eight anchor points that allow for scaling and rotating the model within both mixed and virtual spaces.
